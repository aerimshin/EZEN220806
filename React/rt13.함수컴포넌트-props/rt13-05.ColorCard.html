CSS로 만든 카드를 컴포넌트로 만들어 본다. 작업 순서 1. 컴포넌트 3개 만들기. Label, Square, Card 1-1. Label 컴포넌트 JSX return ( <div className="label"> {props.color} </div> ) 1-2. Square 컴포넌트 JSX let style = { backgroundColor: props.color } return ( <div className="square" style={style}> </div> ) 1-3. Card 컴포넌트 JSX ==> 한번에 모든 속성을 넘기는 방법. ... (중략) return ( <div className="card"> <Square color={props.color}></Square> <Label color={props.color}></Label> </div> ) 또는 return ( <div className="card"> <Square {...props}></Square> <Label {...props}></Label> </div> ) props 가 아래와 같다고 했을 때 { a:"1", b:"2", c:"3" } ...props을 사용하면 props 있는 값들이 풀려서 들어가게 된다. <Square {...props}></Square> 라고 했을 때 ...props을 사용하면 <Square a="1" b="2" c="3"></Square> 처럼 변환되어 실행된다. 2. ReactDOM 을 사용하여 화면에 붙인다. 예시) ReactDOM.render( <div> <Card color={"#ff6663"}></Card> <Card color={"#ffa737"}></Card> </div> , document.getElementById("app") )
